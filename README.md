# AI-Powered Candidate Intelligence & Recruitment Analytics Platform

## Overview
This project automates resume parsing, candidate ranking, and recruitment analytics using Python, SQL, NLP, and Power BI.

## Features
- Resume Parsing
- Candidate Ranking
- NLP-based Job Matching
- ETL Pipelines
- Recruitment KPI Dashboards
- Streamlit Interface

## Technologies Used
- Python
- SQL
- Power BI
- Streamlit
- NLP
- Scikit-learn

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/app.py
